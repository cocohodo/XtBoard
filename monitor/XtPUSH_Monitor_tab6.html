📘 사용설명

  • 초기 동작: 페이지 열면 자동으로 조회가 시작됩니다. (5초 간격)

  • send: 데이터 수신을 시작하거나 멈출 수 있습니다.
    • - 1 ~ 9999(단위 : 초) : 해당 숫자(단위 : 초) 마다 데이터 수신합니다.
    - 0 (단위 : 초) : 데이터 수신을 멈춥니다.

  • 최종 수신 시간: 마지막으로 서버에서 데이터를 받은 시간입니다.

  • 연결 상태: 웹소켓 연결 상태를 보여줍니다 (Connected/Disconnected)
